# 基于Raft的分布式账本

简介：本项目结合 **Raft 共识协议**与**区块链链式存储结构**，构建了一套具备强一致性与防篡改特性的分布式资产管理系统。利用 **ECDSA 非对称加密**与哈希校验保障交易安全，支持创世发币、原子转账、余额审计及风险冻结等核心业务。系统通过 Raft 状态机确保了多节点环境下的数据 **线性一致性**，并通过创世区块权限模型实现了对链上资产流转的完备治理，是一个兼具高可用性与区块链安全特性的金融后端原型。

## 角色及功能介绍

### 创世者

功能：

- 铸币权（只可给管理员转）
- 负责管理员系统
- 审计全网流水

## 管理者

功能：

- 负责注册系统
- 负责充值系统
- 冻结用户资金

- 审查旗下用户流水

### 用户

功能：

- 账户充值

- 点对点转账
- 查询个人流水
